## ✨ 核心特性 (V2.0 Highlight)

### 🖥️ 桌面级生产力布局
- **分栏设计**：左侧时间轴导航，右侧沉浸式书写，参考 TickTick 的高效交互。
- **超长跨度**：支持 2020 - 2099 年的时间穿梭。

### 🛡️ 企业级数据安全 (Local-First)
- **无感云同步**：智能识别 OneDrive / iCloud 路径。只要开启系统同步，数据自动上云，设备损坏也不怕。
- **隐私至上**：所有数据掌握在用户自己手中，无第三方服务器上传。

### 📊 量化自我
- **习惯打卡**：内置习惯追踪系统，支持可视化进度条。
- **数据统计**：通过 Recharts 生成多维图表，直观展示你的心流状态与完成率。

### 🎨 治愈系 UI
- **新拟态风格**：温暖的米色调与柔和阴影。
- **离线可用**：内置 Tailwind CSS 与字体包，无网络环境下依然完美渲染。

## 🛠️ 技术栈

- **Core**: React 19, TypeScript, Vite
- **Desktop**: Electron 39
- **UI**: Tailwind CSS, Lucide Icons, Recharts
- **Storage**: Node.js File System (fs) + LocalStorage

## 🚀 开始使用

1. 在 [Releases](这里填你仓库的release链接) 页面下载最新的 `.exe` 安装包。
2. 安装后即可使用，数据将自动保存至您的【文档】文件夹。
